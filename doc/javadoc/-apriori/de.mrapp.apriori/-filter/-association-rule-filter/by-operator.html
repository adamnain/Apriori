byOperator

@JvmOverloads fun byOperator(operator: Operator, minPerformance: Double, maxPerformance: Double = Double.MAX_VALUE): AssociationRuleFilter

Sets the minimum and maximum performance according to a specific metric, which must be reached by an association rule to be accepted by the filter.

Parameters

operator - The operator, which should be used to calculate the performance of an association rule

minPerformance - The minimum performance, which should be set. The minimum performance must be at least 0

maxPerformance - The maximum performance, which should be set. The maximum performance must be at least the minimum performance
