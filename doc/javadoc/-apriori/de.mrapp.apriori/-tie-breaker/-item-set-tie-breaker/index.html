 fun custom(comparator: Comparator<ItemSet<*>>): ItemSetTieBreaker

Adds an additional custom criteria to the current tie-breaking strategy. The given comparator must return 1, if the first association rule should be preferred, -1, if the second one should be preferred or 0, if no decision could be made.

preferLarge

 fun preferLarge(): ItemSetTieBreaker

Adds an additional criteria to the current tie-breaking strategy, which prefers large item sets over small ones. I.e., if an item set contains more items than its counterpart, it is sorted before the other one.
