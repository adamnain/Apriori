byOperator

fun byOperator(operator: Operator): AssociationRuleTieBreaker

Adds an additional criteria to the current tie-breaking strategy, which performs tie-breaking according to a specific operator. I.e., if an association rule reaches a greater performance according to the given operator, it is sorted before the other one.
