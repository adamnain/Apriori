fun execute(iterable: Iterable<Transaction<ItemType>>): Output<ItemType>

Executes the Apriori algorithm on a specific set of transactions in order to learn association rules, which specify frequent item sets.

Parameters

iterable - An iterable, which allows to iterate the transactions

Return
The rule set that contains the association rules, which have been learned by the algorithm, or an empty rule set, if no association rules have been learned
